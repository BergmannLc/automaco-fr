Este é um projeto de automação do processo Fora de Rota desenvolvido para a empresa Fast Broker Nestlé. Neste projeto, utilizamos algumas ferramentas para abrir o Whatsapp Web, recolher mensagens, alterar planilhas,
analisar e dar resultados com base na localização de clientes e por último devolver o resultado gerado pelo Whatsapp, tudo 100% automatizado.

Utilizamos as seguintes ferramentas:
- Python (com diversas bibliotecas)
- ChromeDriver
